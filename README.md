# Instalaciones requeridas

1. Docker
2. Nest CLI

```
npm i -g @nestjs/cli
```

# Ejecutar en desarrollo

1. Clonar repositorio
2. Instalar dependencias

```
npm install
```

3. Levantar la base de datos

```
docker compose up -d
```

# Tecnologias usadas

- MongoDB
- NestJS
